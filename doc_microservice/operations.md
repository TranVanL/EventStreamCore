## ğŸ’¾ Backup & Restore

### Backup

```bash
# Stop accepting writes (optional for consistency)
curl -X POST localhost:8080/admin/pause

# Backup Raft log and snapshots
tar -czf backup-$(date +%Y%m%d).tar.gz /var/lib/eventstream/raft

# Resume
curl -X POST localhost:8080/admin/resume
```

### Restore

```bash
# Stop node
systemctl stop eventstream

# Clear existing data
rm -rf /var/lib/eventstream/raft/*

# Restore backup
tar -xzf backup-20260131.tar.gz -C /

# Start node
systemctl start eventstream

# Verify
curl localhost:8080/health/ready
```

  -d '{"workers": {"count": 8}}'

### Drain Node

```bash
# Kubernetes

curl -X POST localhost:8080/admin/drain
# Wait for queues to empty
curl localhost:8080/status | jq '.queues'
```

### Force Leader Election

```bash
# Step down current leader
curl -X POST localhost:8080/admin/raft/stepdown
```

### Clear Dead Letter Queue

```bash
# List DLQ events
curl localhost:8080/admin/dlq | jq '.count'

# Retry all
curl -X POST localhost:8080/admin/dlq/retry

# Purge (permanent delete)
curl -X DELETE localhost:8080/admin/dlq
```
